# 更新说明

## 0.11.33 - 2026-06-17

- 聊天角色 Skill 新增“知识博主”和“短视频编导”，总数扩展为 8 个泛化表达风格。
- 每个角色补充适用场景和角色边界，AI prompt 会同时带入角色说明和“不能冒充具体真人达人”的约束。
- 聊天窗角色区改为 4 列两行按钮，并增加当前角色说明卡；说明卡按窗口宽度动态换行，避免长说明挤在一起。
- 聊天窗改为底部输入区固定可见的网格布局，历史区伸缩，快捷按钮和输入框不再被角色区或历史区挤出窗口。
- 最小窗口高度调整为 520x660，补充全窗截图验收，确保输入框和发送按钮可见。
- 角色 Skill 只改变表达方式，不覆盖宠物身份、主人关系和真实故事边界。

## 0.11.32 - 2026-06-17

- 联网资料查询新增中文维基摘要补强，普通知识问题会优先尝试拿到更完整的百科摘要。
- 搜索来源扩展为 DuckDuckGo Lite、Bing 兜底、中文维基摘要和中文维基 OpenSearch，并合并去重。
- 无 AI 时的本地资料回复会显示来源类型和 URL，方便确认资料出处。
- 查询词清洗补充“什么是 / 谁是 / 是什么 / 是谁 / 什么意思”，让普通问句更容易转成可检索实体词。

## 0.11.31 - 2026-06-17

- 修复形象页家人形象列表右侧操作列过窄的问题，“切换/编辑资料”不再被挤成竖排文字。
- 每张形象卡片把操作按钮放到内容区底部，按钮使用稳定宽度；分类、照片和动作数量信息过长时会截断。
- 移除默认宠物代码里的历史本机图片路径，公开包继续不携带本机路径、API Key、聊天、待办或提醒历史。
- 补充 900px 宽度形象页截图验收，后续 UI 改动必须检查按钮、chip 和长说明不重叠、不竖排、不横向溢出。

## 0.11.30 - 2026-06-17

- 新增宠物和编辑资料支持细分类：犬类、猫类、兔类、小型哺乳、大型哺乳、鸟类、爬行动物、水生生物、昆虫/节肢、人物、幻想生物、机器人、拟物/物件和其他。
- 形象页会显示当前宠物分类；新宠物导入和资料编辑可以切换分类 chip。
- 五个基础动作槽位仍兼容旧 atlas，但提示词和界面会按分类解释动作语义，例如人物按站立/走路/招手理解，鸟类按跳步/拍翅理解。
- AI 宠物上下文和生图提示词加入分类、示例和动作方向，避免后续非猫狗形象被写成同一种动作逻辑。

## 0.11.29 - 2026-06-17

- 聊天窗新增“角色 Skill”选择区，可在蛋黄本色、技术导师、产品拆解、研究助手、直说教练、运营写手之间切换。
- 角色 Skill 会写入本机设置，后续 AI 回复自动按当前风格表达，同时保留蛋黄是家人的底层身份。
- 当前角色是能力风格 preset，不冒充具体真人达人，后续可继续扩展为本地 skill 或自定义角色。

## 0.11.28 - 2026-06-17

- 聊天新增通用知识问题识别：“什么是、谁是、为什么、怎么、如何、多少、哪里、哪年、有哪些”等提问会自动走资料查询。
- AI 可用时先查资料再交给模型回答；AI 未配置时返回本地网页摘要，减少普通问题答不上来的情况。
- 搜索层新增中文维基百科 OpenSearch 备用来源，并和 DuckDuckGo Lite 结果去重合并。
- “你是谁、你叫什么、想你、陪我、摸摸”等陪伴语境继续走宠物回复，不误触发网页查询。

## 0.11.27 - 2026-06-17

- 聊天窗快捷区改为 4 列网格，两行展示常用陪伴、时间、待办、查资料和背景入口。
- 工具入口不再依赖单行横向排列，窗口缩窄或后续增加快捷项时不容易溢出。

## 0.11.26 - 2026-06-17

- 聊天新增本地待办查询：问“今天有什么安排”“待办有哪些”“查看一下今天待办”时直接读取本机待办。
- 查询待办优先于新增待办，但“提醒我明天 9 点开会”这类创建意图仍会正常新增提醒。
- 待办查询回复显示标题、时间、分类、优先级和总览统计，AI 未配置时也能回答日程类问题。
- 聊天窗快捷区新增“待办”按钮，并缩短快捷按钮宽度避免横向溢出。

## 0.11.25 - 2026-06-17

- 聊天窗顶部新增能力状态条，展示云端、时间、资料、待办和本地兜底状态。
- 云端能力会按 AI 开关和 Key 配置显示已连接、待配置或已关闭；资料能力会提示当前是查后交给 AI 还是本地网页摘要。
- 待办状态显示未完成和今日数量，聊天创建本地待办后会刷新。
- 聊天窗关闭时清理状态条引用，避免重复打开后残留旧控件。

## 0.11.24 - 2026-06-17

- 聊天新增本地时间工具：问“现在几点/今天几号”时直接使用本机时间回复。
- 联网查询触发词扩展到上网查、联网查、实时、最近、天气、股价、汇率等；AI 未连接时返回网页摘要兜底。
- AI 快速回复补入当前时间和本地待办摘要，让日常安排、待办和资料类对话更顺。
- 聊天窗快捷区新增“时间”“查资料”；AI 页资料查询说明补充本地时间和无 AI 兜底。
- 控制面板保存后显示暖色 Toast，确认设置已写入 E 盘运行配置。
- Windows 免 Python 打包脚本补充应用图标导出，并在 PyInstaller 构建时使用 `danhuang-app-icon.ico`。

## 0.10.0 - 2026-05-24

- macOS 导出改为 Windows 触发 GitHub Actions `macos-latest` 构建 `.app`，完成后下载 `DanhuangDesktopPet-macOS-<version>.zip`。
- 导出弹窗支持配置默认导出目录、可选待办/故事/AI 非敏感配置，并保持 API Key 不导出。
- 新增“设置”页，集中管理默认导出目录、上传目录、图片目录和 GitHub macOS 构建配置。
- 故事编辑器支持拖入写作背景、图片素材插入正文标记，角色提示词查看窗口支持滚动。
- 新宠物和动作生图提示词强化主像素图基准、帧数、尺寸、连贯性、透明边缘和身份一致性要求。
- 动作上传增加追蝴蝶专项 QA：检测多余蝴蝶/碎片、脚底基线和体积漂移；异常追蝴蝶条运行时回退到稳定跑步帧。
- 新增提示词使用教学，说明主像素图、基础动作、单行动作修复和扩展动作的正确使用顺序。
- macOS 导出说明补充目标仓库 `owner/repo`、workflow、分支和 GitHub Token 权限配置。

## 0.9.0 - 2026-05-24

- AI 页改为顶部厂商下拉和当前厂商详情配置，修复厂商推荐文案串到其他 provider 的问题。
- 普通陪聊改为自然语言优先，不再要求 AI 返回 JSON；异常 JSON 回复会清洗后再展示。
- 明确“查一下、最新、资料、新闻、往年今日”等意图时才触发本地资料查询，再交给当前 AI 回复。
- 小米 MiMo 控制台改为平台控制台入口，API Base URL 明确标注不是网页控制台。

## 0.8.0 - 2026-05-23

- 新增“故事”页，支持按宠物记录故事、日记和思念，并生成故事摘要。
- AI 聊天注入当前宠物档案、故事摘要、角色提示词、最近故事和最近对话，避免跨宠物串记忆。
- 聊天窗改为微信式左右气泡，支持每个宠物独立聊天背景和透明度。
- 档案页按月份和日期归档聊天记录，减少长列表滚动。

## 0.7.0 - 2026-05-22

- AI Key 保存改为 DPAPI 本机加密，保存后只显示 `********`，不提供复制或显示真实 Key。
- 模型选择改为下拉，避免模型按钮横向铺开。
- 测试连接只验证当前 provider/model 能返回文本，不要求 JSON。
- 控制面板滚动区域统一重置，减少切页后出现大块空白。

## 0.6.0 - 2026-05-21

- 多宠物档案、等级、聊天和长期记忆改为按宠物隔离。
- 右键快捷菜单增加切换形象、控制面板、待办提醒和 AI 配置入口。
- 新增宠物导入支持主像素图、参考照片和五个基础动作条。
