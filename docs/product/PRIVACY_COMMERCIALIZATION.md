# 隐私和商业化设计

## 隐私边界

默认本地优先：

- 宠物照片、纪念故事、聊天、待办和提醒历史默认只保存在本机。
- 上传生成前必须说明用途并二次确认。
- 用户可导出、备份和删除自己的数据。

不进入公开包：

- API Key、Token、DPAPI 加密内容。
- 聊天记忆、长期记忆、待办、提醒历史、陪伴状态。
- 本机路径、日志、临时文件。
- 未确认授权的真实宠物照片和故事。

## 表达风险

- 纪念宠物不能宣传为“复活宠物”。
- AI 不能承诺医疗、心理治疗、宗教或超自然效果。
- 自动交流应克制，避免情绪操控。

## 免费版

- 默认宠物。
- 基础动作。
- 本地提醒。
- 少量 AI 对话。
- 本地数据存储。

## 付费版

- 真实宠物生成。
- 多宠物家庭。
- 高级动作包。
- 云同步。
- 更多 AI 额度。
- 纪念空间。

## 增值项

- 节日动作包。
- 主题皮肤。
- 气泡主题。
- 桌面小组件。

## 发布前检查

- 隐私协议和服务协议。
- 清晰说明本地保存和上传生成边界。
- 公开包扫描敏感字段。
- 干净 Windows 用户安装验证。
- 杀毒误报检查。
- 崩溃日志脱敏。
