## Review Notes

- `compute_popup_position()` 保持旧行为：优先放在 anchor 右侧，右侧空间不足时翻到右键面板左侧，并按 monitor 边界夹取。
- `right_menu_popup_layout()` 承载 `gap`、`screen_margin`、`close_delay_ms` 和 `max_visible_actions`，减少单文件中的硬编码。
- 两个 Tk 弹层共用 `place_quick_menu_popup()`，降低后续继续拆样式/定位时的重复维护成本。
- `validate_phase3.py` 已输出 `popup_layout`，设计工具和 WPF 后续可以直接读取弹层时序和间距规则。
- README 已补充当前 E 盘版本的启动命令、可见调试命令和进程确认命令。

## Conclusion

Approved。下一步建议继续抽 right-menu 行样式 token 和面板尺寸约束，或转向气泡绘制/预览拆分。
