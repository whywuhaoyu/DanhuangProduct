## 当前运行入口

推荐启动 E 盘镜像：

```powershell
PowerShell -ExecutionPolicy Bypass -File "E:\ProgrammingAlgorithm\VSCodeProjects\DanhuangProduct\data-dev\current-runtime\danhuang\start-danhuang-desktop-pet.ps1"
```

确认当前运行进程：

```powershell
Get-CimInstance Win32_Process | Where-Object { ($_.Name -in @("python.exe","pythonw.exe")) -and ($_.CommandLine -like "*run-danhuang-desktop-pet.py*") } | Select-Object ProcessId,Name,CommandLine
```

## 打包规则

每次优化结束必须做：

1. 同步 `src-prototype/legacy-monolith/run-danhuang-desktop-pet.py` 到 `data-dev/current-runtime/danhuang/run-danhuang-desktop-pet.py`。
2. 运行 `py_compile`、modular 单元测试和 Phase 验证脚本。
3. 用当前 E 盘运行镜像生成 Windows 免 Python exe 包。
4. 清理包内聊天、待办、提醒历史、陪伴状态、API Key、DPAPI blob、本机路径和日志。
5. 产出 package folder、zip 和 SHA256。
6. 记录到 `CHANGELOG.md` 和 `docs/product/执行追踪.md`。
7. 新包验证通过后，只保留最新一组 package folder + zip，删除旧 `danhuang-desktop-pet-windows-*` 打包记录。
